# final task of machine learnning in JN University
## the description of the task
人脸图像性别分类
本次大作业将提供20000多张已经分割的人脸图像，要求基于人脸图像自动识别该人性别。数据集的年龄从1岁覆盖到100多岁，包括了白种人、黄种人、黑种人等多种种族数据。数据集存在人脸姿态、光照、年龄等多种干扰，具有一定的挑战性。
